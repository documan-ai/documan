# Documan.ai

Documentation tools suck. Docusaurus, VitePress, and others need Node.js and constant maintenance. Confluence is where docs go to die. And none of them talk to your AI.

Write simple Markdown. Get a beautiful documentation site with search, AI chat, and a built-in MCP server — your AI can generate, maintain, and search your docs automatically. Perfect for ADRs, business logic, onboarding, and team knowledge bases.

One command to start. AI skill to help you write. CI/CD linter to keep it clean. Self-hosted and free up to 100 MD files.

**Get started:** [docs.documan.ai](https://docs.documan.ai)
